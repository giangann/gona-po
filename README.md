# Live: https://giangann.github.io/gona-po/
# Start in local:<br>
1. yarn install<br>
2. yarn dev<br>

# Deploy:<br>
1. .env => VITE_BASE_URL=/&lt;YOUR_REPO_NAME&gt;<br>
2. package.json => "homepage":"https://&lt;YOUR_GITHUB_USER_NAME&gt;.github.io/&lt;YOUR_REPO_NAME&gt;/"<br>
2. yarn add gh-pages<br>
3. npm run deploy<br>

Done!
